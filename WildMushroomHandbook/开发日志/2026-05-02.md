### 1.2 本地图像抠图 + 白边贴纸功能

实现了核心视觉功能：拍摄照片后本地抠图并生成贴纸效果。

**修改文件**：
- `Views/Camera/CameraView.swift` - 添加照片确认预览和处理流程

**处理流程**：
```
拍摄照片 → 确认预览 → [重拍/确认] → 抠图移除背景 → 添加白边+投影 → 生成贴纸 → Mock识别
```

### Vision 框架背景移除

```swift
// 使用 VNGenerateForegroundInstanceMaskRequest 移除背景
let request = VNGenerateForegroundInstanceMaskRequest()
let handler = VNImageRequestHandler(cgImage: cgImage, options: [:])
try handler.perform([request])

// 生成遮罩图像
let maskedImage = try mask.generateMaskedImage(
    ofInstances: mask.allInstances,
    from: originalImage,
    croppedToInstancesExtent: false
)
```
